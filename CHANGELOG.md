# Change Log

## 1.17.0

Initial release of **Slint AI** — a fork of the official Slint VS Code extension v1.17.0.

### Highlights / 亮点

- **Copy node hierarchy** — copy the full element path from live preview for precise AI targeting (Cursor, Copilot, Claude, etc.)
- **拷贝节点层级** — 从实时预览复制完整 UI 层级路径，让 AI 精准定位要修改的控件
- **Chinese preview UI** / **预览中文界面**
- Desktop bundle with `slint-lsp` (Windows x64)
