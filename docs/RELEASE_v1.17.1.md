# Release v1.17.1

## Node Path Copier for Slint v1.17.1

Community Slint VS Code extension — **not** an official SixtyFPS product.  
Official: [Slint.slint](https://marketplace.visualstudio.com/items?itemName=Slint.slint)

### Changes

- Renamed from "Slint AI" to **Node Path Copier for Slint**
- New distinct icon (hierarchy tree + clipboard, not based on Slint logo)
- Description updated: community extension, not affiliated with SixtyFPS

### Features (unchanged)

- Copy node hierarchy from live preview for AI-assisted editing
- Chinese preview UI (`slint.preview.language`)
- Syntax highlighting, completion, go-to definition, live preview

### Install

**VSIX:**

```sh
code --install-extension slint-ai-1.17.1.vsix
```

**Marketplace:** search `Node Path Copier for Slint` (ID: `slint-ai.slint-ai`)

### Assets

Attach `slint-ai-1.17.1.vsix` from the project root when creating this GitHub Release.
